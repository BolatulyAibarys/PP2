 # Music Player Application

## Description
This project is a simple music player built using Python and Pygame.  
It allows the user to play, pause, and switch between audio tracks using keyboard controls.

## Features
- Play music tracks
- Pause and resume playback
- Switch to next track
- Switch to previous track
- Supports multiple audio files
- Simple keyboard-based control

## Controls
- SPACE — pause / resume music
- RIGHT ARROW — next track
- LEFT ARROW — previous track

## Technologies Used
- Python
- Pygame (pygame.mixer for audio playback)
