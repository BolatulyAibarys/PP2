# Mickey's Clock Application

## Description
This project is a digital-style clock built using Python and Pygame.  
It displays the current system time using graphical clock hands inspired by Mickey Mouse.

## Features
- Displays current system time (minutes and seconds only)
- Clock updates in real-time (every second)
- Two graphical hands are used:
  - Minute hand (represents minutes)
  - Second hand (represents seconds)
- Smooth rotation based on system time
- Uses image rotation with Pygame

## Technologies Used
- Python
- Pygame
