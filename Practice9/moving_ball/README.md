# Moving Ball Game

## Description
This project is a simple interactive game built with Python and Pygame.  
A red ball can be moved across the screen using the arrow keys.

## Features
- Displays a red ball on a white background
- Ball moves with arrow keys
- Each key press moves the ball by 20 pixels
- Ball cannot leave the screen boundaries
- Invalid moves outside the screen are ignored

## Technologies Used
- Python
- Pygame

1. Install Pygame:
pip install pygame

2. Run the program:
python main.py
